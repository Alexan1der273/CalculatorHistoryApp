# Калькулятор с историей операций

Windows Forms приложение на C# (.NET Framework 4.7.2 / .NET 8) – базовый калькулятор с сохранением истории в JSON.

## Возможности
- Операции: `+`, `-`, `*`, `/`, `%`
- Цифры, десятичная точка, `C` (очистить), `⌫` (удалить символ), `±` (смена знака)
- Результат в текстовом поле
- История: выражение, результат, дата и время – в отдельном списке
- Сохранение в `history.json`, загрузка при запуске

## Сборка и запуск (локально)

1. Создайте проект Windows Forms (.NET Framework 4.7.2 или .NET 8).
2. Замените `Program.cs`, `Form1.cs`, `Form1.Designer.cs` кодом из репозитория.
3. Установите NuGet-пакет `Newtonsoft.Json`.
4. Соберите и запустите (F5).

## Пример работы

Ввод: `123 + 45` → `=` → результат `168`  
История: `123 + 45 = 168 [2025-03-31 15:00:00]`
